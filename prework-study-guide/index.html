HTML

  • The is the framework of developing a website and makes it easy for users to navigate.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class attributes allows us to share a css rule to any element that is chosen.
  • External style sheets is the most common way to style a webpage

Git

  • git checkout -b branch-name: creates a new branch and switches to it
  • version control: tracks changes to a file or files so you can recall a spefici version later
  • git clone: cloning an existing git repo
